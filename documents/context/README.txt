ConfAI Context Files
====================

This folder contains context files that are automatically injected into the AI's system prompt for every conversation.

How to use:
-----------
1. Upload .txt or .md files through the Admin Dashboard
2. Keep each file under 500KB
3. Files are concatenated and added to the system context
4. Changes take effect immediately for new conversations

Best practices:
--------------
- Use clear, descriptive filenames
- Keep content focused and relevant
- Update files as conference information changes
- Monitor context window usage in the admin dashboard

Example use cases:
-----------------
- Conference schedule and venue information
- Speaker bios and session descriptions
- FAQs and common questions
- Product information or announcements
- Guidelines for AI responses
